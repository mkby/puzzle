# 影像大师 Pro (Photo Master Pro)

一个集成了**智能拼图**与**专业边框**功能的现代化 Web 图像处理工具。无需安装，打开浏览器即可使用，支持高清无损导出。

## ✨ 核心功能

### 🧩 拼图大师 (Puzzle Master)
- **智能布局**：支持 3-12 张图片自动生成最优网格布局。
- **自由调节**：内置 1:1, 4:3, 16:9 等常用比例，支持自定义比例。
- **交互操作**：拖拽平移图片，滚轮缩放大小。
- **高清导出**：生成 2400px 宽度的高清拼图。

### 🖼️ 边框大师 (Frame Master)
- **批量处理**：一次性上传多张照片，自动添加边框。
- **参数显示**：自动读取并显示照片的 EXIF 信息（相机型号、焦距、光圈、快门、ISO）。
- **风格定制**：调节边框宽度、背景模糊程度和文字大小。
- **一键打包**：支持将生成的所有图片打包为 ZIP 下载。

## 🚀 使用方法

1. **启动**：直接使用浏览器打开 `index.html`。
2. **切换模式**：在左侧（或顶部）侧边栏点击“拼图大师”或“边框大师”进行切换。
3. **拼图模式**：
   - 上传 3-12 张图片。
   - 选择画布比例，调整边框粗细和颜色。
   - 调整图片位置和大小，点击“下载高清拼图”。
4. **边框模式**：
   - 批量上传照片。
   - 调整边框宽度、背景模糊度和 EXIF 显示设置。
   - 点击“开始生成”预览效果，满意后点击“打包下载全部”。

## 🛠️ 技术栈

- **核心**：HTML5, CSS3, Vanilla JavaScript
- **样式**：Tailwind CSS
- **库支持**：
  - `exif-js`: 读取照片元数据
  - `jszip`: 打包下载
  - `FileSaver.js`: 文件保存
- **字体**：Inter, Oswald, Roboto Mono
