# 影像大师 (Photo Master)

多功能Web图像处理工具。无需安装，打开浏览器即可使用。

## ✨ 核心功能

### 🧩 拼图大师
- 智能拼图布局，支持多种预设比例（1:1, 3:4, 9:16等）
- 拖拽排序，自由调整图片位置
- 自定义边框颜色和宽度
- 实时预览，高质量导出

### 🖼️ 边框大师
- 批量处理多张图片
- 自定义边框宽度和样式
- 实时预览效果

### 💧 水印大师
- 多种水印类型：文字、图片、色板、直方图
- 智能色板提取
- RGB/HSL 直方图生成
- 自由调整水印位置和大小

### 🏷️ 标签大师
- AI 智能识别图片内容
- 自动生成相关标签
- 支持批量处理
- 多语言输出（中文/英文/自动检测）

### ✍️ 文案大师
- AI 生成小红书风格文案
- 支持多种风格：通用、种草、摄影、教程、日常
- 一键生成标题和正文
- 快速复制分享

## 🛠️ 技术栈

- **HTML5 Canvas**: 高性能图像处理
- **Tailwind CSS**: 现代化 UI 设计
- **Vanilla JavaScript**: 轻量高效，无框架依赖
- **Font Awesome**: 图标库
- **AI APIs**: OpenAI GPT-4 Vision, Google Gemini

## 🔒 隐私说明

- 图片处理完全在浏览器本地完成
- 仅在使用 AI 功能时会将图片发送到配置的 API 服务
- API 密钥存储在浏览器 localStorage，不会上传到任何服务器
- 建议使用自己的 API 密钥以确保数据安全
